# 图片压缩工具 — 质量滑块提示优化

## Context

用户发现 quality=100 压缩后图片反而变大。这是因为 quality=100 表示「最高质量、最少压缩」，不是「压缩到最小」。需要在质量滑块下方添加提示文字，帮助用户理解。

## 修改内容

**文件**：`sc-frontend/src/app/tools/compress/page.tsx`

在质量滑块的数字输入框下方添加一行灰色提示文字：

```
数值越高质量越好，但文件越大；100 可能比原图更大
```

仅此一处前端修改，无后端变更。

## 验证方式

手动访问 `/tools/compress`，确认滑块下方提示文字显示正确。
